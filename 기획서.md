# 프로젝트 기획서: Real-time Flowchart for VS Code

## 1. 프로젝트 개요

### 1.1. 프로젝트명
- (가칭) Realtime Code Flowchart for VS Code / Cursor Extension

### 1.2. 목표
- 작성 중인 코드의 `if-else`, `switch` 문 등 조건/분기 로직을 실시간으로 분석하여 시각적인 순서도(Flowchart)로 표현한다.
- 개발자가 복잡하게 꼬인 로직을 직관적으로 이해하고, 잠재적인 버그를 조기에 발견할 수 있도록 돕는다.
- 주니어 개발자 또는 로직 파악이 어려운 코드베이스에 대한 학습 및 이해를 돕는 시각화 도구를 제공한다.

### 1.3. 핵심 기능
- **실시간 분석:** 코드 에디터에서 함수 작성 시 `if-else`, `switch` 문을 자동으로 감지 및 분석.
- **순서도 생성:** 분석된 로직을 기반으로 직관적인 순서도(Flowchart)를 VS Code/Cursor 내 별도 패널 또는 뷰에 렌더링.
- **강조 기능:** 현재 커서 위치에 해당하는 코드 블록의 순서도 노드를 하이라이트. (추가 고려)

## 2. 타겟 사용자

- 로직이 복잡한 함수를 작성하거나 분석해야 하는 개발자
- 새로운 코드베이스의 흐름을 빠르게 파악하고자 하는 개발자
- 특히 주니어 개발자 또는 시각적인 자료에 익숙한 사용자

## 3. 기술 스택 (예상)

- **언어:** TypeScript (VS Code Extension 개발 표준)
- **프론트엔드 (순서도 렌더링):** Webview API + D3.js, React Flow, Mermaid.js 등 (순서도 라이브러리 선택 필요)
- **코드 분석:** VS Code Language Server Protocol (LSP) 활용 또는 Tree-sitter 같은 파서 라이브러리 고려. (실시간 AST 분석)

## 4. MVP (Minimum Viable Product) 범위

- 특정 함수 내 `if-else` 및 `switch` 문에 대한 순서도 기본 생성 및 렌더링 기능 구현.
- 지원 언어: (초기에는) JavaScript/TypeScript 먼저 지원.
